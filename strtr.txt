нужно что бы получилось красивое полноценное и функциональное мини приложение для телеграмм и веб версии для ПК (что б можно было с ПК использовать). 
приложение должно быть с современным дизайном (как в современных мобильных приложениях), логотипом (картинкой) при запуске, кнопками, меню. 
Должна быть обязательно у всех ролей кнопка "выход" - при ее назатии должна завершиться сессия пользователя и приложение должно перейти на первый экран (авторизацию по паролю заново)

Название приложения:
AWR 
assistant work resurs (новая строчка)

Суть приложения: 
Есть супер-админ (1 человек) 
Есть админы ( 2 человека) 
Есть бригады (2 бригады по 2 чел.) 
Есть кладовщик (1 человека)

Афторизация: 
пользователь открывает бот, нажимает поледиться номером, если номер есть в базе то бот дает кнопку запустить приложение, если номера нет в базе - то пишет в доступе отказано. 
После того как бот принял номер и пользователь нажал запустить приложение - На стартовом экране открывается логотип + пункт авторизоваться (введите пароль  пароль (вноситься в базу).  Пользователь авторизируется и, если он есть в базе то ему открывается мини приложение с его интерфейсом в зависимости кем он является (админ, бригада, кладовщик) 

У каждой роли свое меню и функии. У всег есть кнопка "Выход" (которая завершает сеанс авторизации и перевадит на главный экран где нужно снова ввести пароль для авторизации)

Роли и меню 
Супер-админ: пункты меню и их функции 
Задача админу - назначает задачи админам 
Новая задача - может назначать задачи бригадам 
Список задач - видят общий список задач (есть фильтр по бригадам и админам, статусам) 
Материала - видят остатки материалов и инструмента по бригадам, и на складе. 
Редактировать задачу - Могут изменить текст задания, переназначить в задании бригаду, удалить задание, менять статус задания.
Отчёты- пункт где хранятся все отчёты бригад и админов и есть фильтры для поиска отчётов по статусу, адресу, бригаде/админу. 
доступ инфо - пункт где хранится инфо о доступе из отчётов бригад/админов за весь период и есть тоже фильтр по адресу.
Приемка - пункт куда вносятся адреса домов поторые сдаются заказчику и так же удаляются, редактируются
Карта - та же карта что и у бригад , только без каких либо отметок
Список бригад - Список бригад которые работают в компании  (есть фильтр "в работе", "на больничном", "в командировке", "в отпуске", "увольняются")

Админы: пункты меню и их функции
Новая задача - может назначать задачи бригадам 
Список задач - видят общий список задач (есть фильтр по бригадам и админам, статусам) 
Материала - видят остатки материалов и инструмента по бригадам, и на складе. 
Редактировать задачу - Могут изменить текст задания, переназначить в задании бригаду, удалить задание, менять статус задания.
Отчёты- пункт где хранятся все отчёты бригад и админов и есть фильтры для поиска отчётов по статусу, адресу, бригаде/админу. 
доступ инфо - пункт где хранится инфо о доступе из отчётов бригад/админов за весь период и есть тоже фильтр по адресу.
Приемка - пункт куда вносятся адреса домов поторые сдаются заказчику и так же удаляются, редактируются  
Карта - та же карта что и у бригад , только без каких либо отметок
Список бригад - Список бригад которые работают в компании  (есть фильтр "в работе", "на больничном", "в командировке", "в отпуске", "увольняются")

задание должно иметь поля:
- Адрес
- колличество этажей и парадных 
- вид работ (воздушка, перемычка, растянуть дом, дом под ключ, бурение, сварки) 
- ТЗ
- Доступ 
- Отметка как "Срочно!" (можно поставить или не ставить)
- Выбор бригады исполнителя. 

Есть статусы для заданий. бригады не могут менять статус задания. 
- Новая задача (не имеет бригаду исполнителя, и только после перевода в статус "в работе" даёт возможность назначить бригаду" 
    - В работе 
    - Выполнено
    - Отложено
    - Проблемный дом 


Бригады: пункты меню и их функции
Мои задачи- видят список задач назначенные именно им в работу, дают по ним отчет - после заявка переходит в статус  "Выполненные задачи"
Выполненые задачи - видят список выполненных задач (не могут редактировать отчет после того как переведено в статус выполнено) 
Отложенные задачи - видят список отложенных задач (без редактиррования, и отчета)
Доступ инфо - пункт где хранится инфо о доступе из отчётов бригад за весь период и есть тоже фильтр по адресу.
Мои материалы - видят остаток материалов что закреплён за ними после списания. 
Мой инструмент - видят инструмент который закреплён за ними.
Карта - на этой карте должны быть отмечены адреса задач "в работе" что назначены именно им

Бригады выполняют задачи, дают отчёт и списывают материал по каждой задаче.
- отчет должен быть из 4 частей, и может заполняться в разное время
1 часть - комментарий 
2 часть - доступ
3 часть - Фото (от 1 до 20 фоток)
4 часть - списать материалы (списывают каждый вид материала в отдельной строчке) 
после заполнения каждого из пунктов отчёта (при сохранении) появляется галочка. 

задача должна оставаться в статусе "в работе" пока не будут сделан отчет по 4 пунктам. после переводиться в статус "выполнено". 


Кладовщик:пункты меню и их функции
Пополнить ТМЦ - добавляет материалы на общий склад 
Редактировать ТМЦ - может списывать материалы со склада (при списании пишет комментарий) 
Пополнить инструмент - добавляет инструмент на общий склад  
Редактировать инстррумент- может списывать инструмент со склада (при списании пишет комментарий)  
ТМЦ - выдает бригадам материалы , может вернуть на склад с бригады, переместить между бригадами 
Инструмент - выдает бригадам инструмент, может вернуть на склад с бригады, переместить между бригадами
ТМЦ бригад- Видит список бригад, у кого сколько материалов на руках 
Инструмент бригад - видит список бригад , у кого какой инструмент
Остатки - видно общие остатки по складу. (материал + инструмент что числится на складе, 2 разные части) 
- ведёт учёт. (видно все перемещения по складу (есть фильтр "ТМЦ", "Инструмент" 
Есть кнопка выгрузить отчёт в excel файл.

Материалы считаются в метрах ,штуках и килограммах (м, шт и кг)

Список материалов и в чем измеряются: 
- Кабель ВОК 4 100(м)
- БО/16 100(шт)
- шпаклёвка 100(кг)

Список Инструмента должен фиксироваться по серийному номеру:
Список:
перфоратор проводной
перфоратор безпроводной  
аккумулятор 6 ампер

Весь процесс должен быть именно в приложении , телеграм бот только фильтрует номера и дает или не дает кнопку открыть приложение  
Все отчеты должны сохраняться в приложении +  отправляться в супергруппу телеграмм где у каждой бригады свой топик.

У всех пользователей должно быть меню с кнопками . 


Приложение буду запускать на рендер через гитхаб
