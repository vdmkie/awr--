помоги создать красивое полноценное и функциональное мини приложение для  телеграмм и что б можно было с ПК использовать с современным дизайном , логотипом (картинкой) при запуске, кнопками, меню. 

Название приложения:
AWR 
assistant work resurs 

Суть приложения: 
Есть админы ( 4 человека) 
Есть бригады (10 бригад по 2 чел.) 
Есть кладовщик 

Админы:
 назначают на бригады задания и видят общий список задач, материалов и инструмента по бригадам, могут переназначить в задании бригаду, удалить задание, менять статус задания.
есть пункт отчёты (где хранятся все отчёты бригад и есть фильтры для поиска отчётов по статусу, адресу, бригаде. 
есть пункт доступ (где хранится инфо о доступе из отчётов и есть тоже фильтр по адресу 

задание должно иметь поля:
Адрес
ТЗ
Доступ 
пометка 
Бригада что назначена. 

Есть статусы: 
Новая задача 
В работе 
Выполнено 
Отложено
Проблемный дом 


Бригады:
видят список задач назначенные им, видят остаток материалов и инструмента что закреплён за ними. 
выполняют задачи, дают отчёт и списывают материал по каждой задаче.
отчет должен быть из 4 частей, и может заполняться в разное время
1 часть - комментарий 
2 часть - доступ
3 часть - Фото
4 часть - материалы

задача должна оставаться в статусе "в работе" пока не будут сделан отчет по 3 пунктам. 


Кладовщик:
добавляет материалы и инструмент на общий склад (так же может убрать материалы и инструмент со склада) выдает бригадам материалы и инструмент, может вернуть на склад с бригады, переместить между бригадами и ведёт учёт...
Видит таблицу со списком бригад, у кого сколько материалов на руках и какой инструмент, видно остатки по складу. 
Есть кнопка выгрузить excel файл на ПК
Материалы считаются в метрах ,штуках и килограммах (м, шт и кг)

пользователь открывает бот, запускает приложение. На стартовом экране пункт авторизоваться  (телеграмм ID + пароль (вноситься в базу).  Пользователь авторизируется и, если он есть в базе то ему открывается мини приложение с его интервейсом в зависимости кем он является (админ, бригада, кладовщик) 

У меня в ТГ есть супергруппа с топиками для каждой бригады.

Весь процесс должен быть именно в приложении , телеграм бот только открывает мини приложение

У всех пользователей должно быть меню с кнопками . 

Интерфейс админа: 
Меню:
- активные задачи (фильтр и редактирование задач) 
- назначить задачу (и все функции ТЗ)
- материалы что у бригад на руках 
- инструмент что у бригад на руках 
- остатки материалов по складу

Интерфейс бригады: 
Меню: 
- мои задачи (показывает активные) 
- выполненные задачи 
- отложенные задачи 
- проблемные дома 
- мои материалы (показывает остатки что на руках у бригады) 
- мой инструмент (показывает какое оборудование у них числится) 

Интерфейс кладовщика: 
Меню:
- выдать материал (на бригаду) 
- принять материал (от бригады) 
- переместить материал (между бригадами) 
- выдать инструмент (на бригаду)
- принять инструмент (на склад) 
- переместить инструмент (между бригадами) 
- пополнить склад (материалы или инструмент которые вносятся на склад) 
- остатки по складу (материал и инструмент что числиться на складе и никому не выдан)
